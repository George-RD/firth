Michael Hicks and Scott M. Nettles, "Dynamic Software Updating", ACM
Transactions on Programming Languages and Systems 27(6), 2005. Their system
uses verifiable native-code patches, dynamic linking, explicit state
transformation, and rollback to support type-safe updates of running programs.
