The observed mathlib development `lean-toolchain` content on 2026-07-17 is
`leanprover/lean4:v4.32.0`. Mathlib's dependency guidance requires a
downstream project to use the same Lean toolchain as mathlib. This is not used
here because it is not the latest stable Lean release verified for this
decision and Firth has no mathlib dependency yet.
