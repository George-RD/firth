Frozen v0.1 specification of the Firth kernel calculus: a typed concatenative
calculus with a single value stack, explicit value and stack typing, prenex
row-polymorphic erased word types, recursive quotation usage meets, finite-trace
at-most-once linearity safety, the linear World effect model, and the
parameterised cost table κ. The kernel dictionary stores no refinements; public
contracts and refinement subsumption belong to the elaborator.
